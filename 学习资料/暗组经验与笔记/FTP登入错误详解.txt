1、 无法连接到远程服务器。Socket 错误 = #10060。 

原因：对方未开FTP或者无法连通   端口错误（默认21）

2、530 Sorry, no ANONYMOUS access allowed 

原因：帐号密码错误或者拒绝登入	

3、Too many users - please try again later. 

原因：限制了登入的人数

4、530 Not logged in, only one session from same IP allowed at a time. 

原因：该帐号只允许单线程登入

5、在登入人数已满，而你不断尝试登入，有三种可能：

      你终于挤进去了
	
      弹出错误提示（控制连接已关闭）表明ftp拦截你这样频繁的尝试

      你被ban了（530 Not logged in, unauthorized IP address.）

　　200　　　 命令成功

　　213　　　 文件状态 
　　214　　　 帮助信息，信息仅对人类用户有用 
　　215　　　 名字系统类型 
　　220　　　 对新用户服务准备好 
　　221　　　 服务关闭控制连接，可以退出登录
